# NurseBnB Property Images

## How to Add Your Property Photos

Download images from your Zillow listing and save them here with these exact filenames:

### Property Images (`/images/property/`)

Save your main property photos here:

| Filename | Description | Suggested Zillow Photo |
|----------|-------------|------------------------|
| `hero.jpg` | Main exterior/waterfront shot | Best exterior view of house |
| `exterior-1.jpg` | Front of house | House from street |
| `exterior-2.jpg` | Side/back of house | Another exterior angle |
| `waterfront.jpg` | Bay/water view | View of Liberty Bay |
| `deck.jpg` | Deck/patio area | Outdoor deck with view |
| `living-room.jpg` | Main living area | Living room interior |
| `kitchen.jpg` | Kitchen | Kitchen view |
| `dining.jpg` | Dining area | Dining room/area |
| `bedroom-1.jpg` | Master bedroom | Main bedroom |
| `bedroom-2.jpg` | Second bedroom (if applicable) | Guest bedroom |
| `bathroom-1.jpg` | Main bathroom | Bathroom |
| `bathroom-2.jpg` | Second bathroom (if applicable) | Second bath |
| `view.jpg` | View from property | Olympic Mountains/bay view |

### Unit-Specific Images (`/images/units/`)

For each unit, save photos with these prefixes:

**Studio Suite:**
- `studio-living.jpg`
- `studio-kitchen.jpg`
- `studio-bed.jpg`
- `studio-bath.jpg`
- `studio-entrance.jpg`
- `studio-patio.jpg`

**Garden Suite:**
- `garden-bedroom.jpg`
- `garden-bath.jpg`
- `garden-shared-kitchen.jpg`
- `garden-entrance.jpg`

**Upper Retreat:**
- `upper-living.jpg`
- `upper-kitchen.jpg`
- `upper-bedroom.jpg`
- `upper-bath.jpg`

- `upper-den.jpg`

## Image Specifications

- **Format:** JPG or WebP preferred
- **Size:** At least 1200px wide for best quality
- **Aspect Ratio:** 16:9 or 4:3 works best

## Quick Start

1. Go to your Zillow listing: https://www.zillow.com/homedetails/20186-Front-St-NE-Poulsbo-WA-98370/23406812_zpid/
2. Right-click on each photo and "Save Image As..."
3. Rename to match the filenames above
4. Drop into the appropriate folder
5. Refresh your browser - images will automatically appear!
